# AccountStatusEnum

Status of the credit account.  *NOTE* `CHARGE_OFF` is not an allowable value for `original_status`.

## Enum

* `Unactivated` (value: `'UNACTIVATED'`)

* `Active` (value: `'ACTIVE'`)

* `Suspended` (value: `'SUSPENDED'`)

* `Terminated` (value: `'TERMINATED'`)

* `ChargeOff` (value: `'CHARGE_OFF'`)
