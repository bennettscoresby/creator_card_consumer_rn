# PolicyRewardAccrualType

Reward accrual strategy for a credit program. If no value for `accrual_strategy` is set, the default value of this field is `DEFAULT`.

## Enum

* `Default` (value: `'DEFAULT'`)

* `Payment` (value: `'PAYMENT'`)
